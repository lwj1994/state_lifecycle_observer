# Example App

This example demonstrates how to use `state_lifecycle_observer` to manage multiple controllers with minimal boilerplate.

## Features Verified

- **TabControllerObserver**: Managing a `TabController` for 3 tabs.
- **AnimControllerObserver**: Managing an `AnimationController` (auto-repeat).
- **ScrollControllerObserver**: Managing a `ScrollController` in a list.
- **TextEditingControllerObserver**: Managing a text field input.

## How to Run

```bash
cd example
flutter run
```
